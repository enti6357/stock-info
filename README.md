# Stock Info - 주식 정보 사이트

Next.js 기반 실시간 주식 정보 제공 서비스

## 기술 스택

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 시작하기

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 실행
npm start
```

## 기능

- 📊 실시간 주식 시세
- 📈 차트 분석
- 🔔 가격 알림
- 🌓 다크모드 지원

## 배포

Vercel을 통해 자동 배포됩니다.
